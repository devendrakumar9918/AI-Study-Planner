# 📚 AI Study Planner

An AI-powered Study Planner that helps students organize subjects, generate personalized study plans using Google Gemini AI, and track their learning progress.

---

## 🚀 Features

- 👤 User Authentication (Signup & Login)
- 🔐 JWT Authentication
- 📖 Subject Management
- 🤖 AI Study Plan Generation using Google Gemini
- 📅 Study Schedule
- 📆 Calendar
- 📊 Progress Tracking
- ⏰ Study Reminders
- 📱 Responsive User Interface

---

## ⚙️ Installation

### Clone Repository

```bash
git clone YOUR_GITHUB_REPOSITORY_LINK
```

### Backend

```bash
cd backend
npm install
node server.js
```

---

## Environment Variables

Create a `.env` file inside backend.

```
PORT=5000
MONGODB_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET
GEMINI_API_KEY=YOUR_API_KEY
```
